## Useful Options

`--tool=<toolName>` By default, this is memcheck
`--log-file=<file>` If terminal output is too big
`--track-origins=yes` Can slow down debugging but tracks origin of uninitialized vals
`--verbose` Verbose output or not - useful for diving in deep

## Useful Flags

`-g` debugging output (use in gcc/g++ compile)
`-0g` debugging output with debug-friendly optimisations (use in gcc/g++ compile)
